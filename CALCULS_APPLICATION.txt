    1. PARAMÈTRES DE CONFIGURATION (SAS)
═══════════════════════════════════════════════════════════════════════════════

L'application utilise des constantes système fixes pour modéliser la réalité 
fiscale d'un dirigeant de SAS (assimilé-salarié) :

┌─────────────────────────────────────────────────────────────────────────────┐
│ VAT_RATE (Taux de TVA)                                                      │
├─────────────────────────────────────────────────────────────────────────────┤
│ Valeur : 0.10 (10%)                                                         │
│ Description : Taux de TVA standard appliqué aux ventes de produits traiteur │
│ Usage : Conversion prix HT ↔ prix TTC                                       │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ CHARGED_HOURLY_RATE (Coût horaire chargé SAS)                               │
├─────────────────────────────────────────────────────────────────────────────┤
│ Valeur : 27.30 €/h                                                          │
│ Calcul : 15.00 €/h (net souhaité) × 1.82 (coefficient de charges globales)  │
│ Description : Coût réel supporté par l'entreprise pour rémunérer le         │
│               dirigeant à hauteur de 15.00 € nets de l'heure.               │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ OVERHEAD_RATE (Frais Généraux)                                              │
├─────────────────────────────────────────────────────────────────────────────┤
│ Valeur : 0.10 (10%)                                                         │
│ Description : Facteur couvrant les frais de fonctionnement indirects.       │
│ Application : Appliqué sur le Prime Cost (Coût Matière + Main d'œuvre)      │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ TARGET_NET_MARGIN (Marge nette cible de l'entreprise)                       │
├─────────────────────────────────────────────────────────────────────────────┤
│ Valeur : 0.25 (25%)                                                         │
│ Description : Pourcentage de marge nette visé pour le bénéfice entreprise   │
│ Usage : Détermination du prix de vente suggéré                              │
└─────────────────────────────────────────────────────────────────────────────┘

    2. CALCULS DE BASE & RÈGLE D'ARRONDI
═══════════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────────────┐
│ RÈGLE D'ARRONDI STRICTE (formatCurrency)                                    │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ Afin de garantir la rentabilité et d'absorber les micro-pertes de           │
│ manipulation, TOUTES les valeurs monétaires affichées sur l'interface       │
│ subissent un arrondi supérieur obligatoire :                                │
│                                                                             │
│ 1. RÈGLE PRINCIPALE : Arrondi à la dizaine de centimes supérieure           │
│    Formule : Arrondi = Math.ceil(Montant × 10) ÷ 10                         │
│    Exemple : 4.01 € devient 4.10 €                                          │
│                                                                             │
│ 2. EXCEPTION (Micro-prix < 0.10 €) : Arrondi au centime supérieur           │
│    Formule : Arrondi = Math.ceil(Montant × 100) ÷ 100                       │
│    Exemple : 0.072 € devient 0.08 € (évite de gonfler à 0.10 €)             │
│                                                                             │
│ IMPORTANT : Les calculs intermédiaires en base de données et en mémoire     │
│ utilisent les valeurs réelles non arrondies pour éviter l'accumulation      │
│ d'erreurs d'arrondi. L'arrondi est appliqué uniquement à l'affichage.       │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ 2.1 - COÛT MATIÈRE D'UNE RECETTE (calculateRecipeCost)                      │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ FORMULE BRUTE :                                                             │
│   Coût matière brut (CM) = Σ (Prix ingrédient × Quantité)                   │
│                                                                             │
│ DESCRIPTION :                                                               │
│   Calcule le coût total brut de tous les ingrédients d'une recette.         │
│   Les ingrédients n'ayant pas de prix spécifié sont ignorés (considérés     │
│   comme valant 0.00 €).                                                     │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

    3. LOGIQUE FINANCIÈRE DE LA MÉTHODE "PRIME COST"
═══════════════════════════════════════════════════════════════════════════════

Pour éviter de faire payer deux fois la main d'œuvre ou de générer des marges
absurdes basées uniquement sur un coefficient, l'application utilise la méthode
standardisée du Prime Cost.

  • Coût Matière (CM) : Food Cost total de la recette.
  • Coût Main d'œuvre (MO) : Labor Cost total (Temps de travail × Taux chargé).
  • Prime Cost : Somme de la matière et du temps de travail (CM + MO).
  • Frais Généraux (FG) : Fixés à 10% du Prime Cost (couvrent énergie, usure).
  • Coût de Revient Complet : Prime Cost + Frais Généraux.

    4. CALCULS DE RENTABILITÉ DÉTAILLÉS
═══════════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────────────┐
│ 4.1 - COÛT MAIN D'ŒUVRE TOTAL (calculateLaborCost)                          │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ FORMULE :                                                                   │
│   Coût MO brut = (Temps production en minutes ÷ 60) × 27.30                 │
│                                                                             │
│ DESCRIPTION :                                                               │
│   Calcule le coût horaire chargé SAS pour le temps passé en cuisine.        │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ 4.2 - COÛT COMPLET PAR PORTION (calculateTotalCostPerServing)               │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ FORMULES :                                                                   │
│   Prime Cost = CM + MO                                                      │
│   FG = Prime Cost × 0.10                                                    │
│   Coût Complet / Portion = (Prime Cost + FG) ÷ Portions                     │
│                                                                             │
│ DESCRIPTION :                                                               │
│   Calcule le coût de revient réel unitaire, toutes charges comprises.       │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ 4.3 - PRIX DE VENTE HT ACTUEL PAR PORTION (calculateActualPriceHTPerServing)│
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ FORMULE :                                                                   │
│   PV HT Actuel / Portion = (CM ÷ Portions) × Coefficient Multiplicateur     │
│                                                                             │
│ DESCRIPTION :                                                               │
│   Le prix de vente réel fixé par l'artisan, basé uniquement sur le          │
│   coefficient multiplicateur appliqué au coût de la matière.                │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ 4.4 - MARGE NETTE RÉELLE (calculateNetMargin)                               │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ FORMULE :                                                                   │
│   Marge Nette (%) = ((PV HT Actuel - Coût Complet) ÷ PV HT Actuel) × 100    │
│                                                                             │
│ DESCRIPTION :                                                               │
│   Pourcentage de gain réel net d'entreprise après déduction de toutes les   │
│   charges (matière, salaire chargé du gérant et frais généraux).            │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ 4.5 - RENTABILITÉ HORAIRE COMPLÉMENTAIRE (calculateHourlyProfitability)     │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ FORMULE :                                                                   │
│   Profit horaire brut = (Profit unitaire × Portions) ÷ (Temps en minutes / 60)│
│   Où Profit unitaire = PV HT Actuel - Coût Complet / Portion                │
│                                                                             │
│ ANALYSE DES RÉSULTATS :                                                     │
│   • >= 0 € : Le traiteur a payé ses matières, ses frais de fonctionnement,  │
│              s'est versé son salaire net de 15 €/h et l'entreprise dégage   │
│              un bénéfice supplémentaire.                                    │
│   • < 0 €  : Le traiteur travaille à un salaire net horaire inférieur à 15 €.│
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ 4.6 - PRIX DE VENTE SUGGÉRÉ (calculateSuggestedPrice)                       │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ FORMULES :                                                                   │
│   PV suggéré HT = Coût Complet / Portion ÷ (1 - Marge cible)                │
│                 = Coût Complet / Portion ÷ 0.75                             │
│   PV suggéré TTC = PV suggéré HT × 1.10 (TVA 10%)                           │
│                                                                             │
│ DESCRIPTION :                                                               │
│   Calcule le prix HT et TTC pour atteindre les 25% de marge nette cible.    │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

    6. FORMULES DÉTAILLÉES ET EXEMPLES COMPLETS
═══════════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────────────┐
│ 6.1 - EXEMPLE COMPLET 1 : MOUSSE AU CHOCOLAT                                │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ DONNÉES DE BASE :                                                           │
│   • Portions : 6  | Temps : 30 minutes (0.5 h)  | Multiplicateur : 3.5      │
│                                                                             │
│ INGRÉDIENTS :                                                               │
│   • Chocolat noir 70% : 200g à 0.015 €/g = 3.00 €                           │
│   • Œufs : 4 pièces à 0.25 €/pièce = 1.00 €                                 │
│   • Sucre : 50g à 0.0015 €/g = 0.075 €                                      │
│                                                                             │
│ ─────────────────────────────────────────────────────────────────────────   │
│ ÉTAPE 1 : COÛT MATIÈRE (CM)                                                 │
│ ─────────────────────────────────────────────────────────────────────────   │
│   CM total brut = 3.00 + 1.00 + 0.075 = 4.075 €      → AFFICHÉ : 4.10 €     │
│   CM unitaire brut = 4.075 ÷ 6 = 0.679167 €          → AFFICHÉ : 0.70 €     │
│                                                                             │
│ ─────────────────────────────────────────────────────────────────────────   │
│ ÉTAPE 2 : COÛT MAIN D'ŒUVRE (MO)                                            │
│ ─────────────────────────────────────────────────────────────────────────   │
│   MO total brut = (30 ÷ 60) × 27.30 = 13.65 €        → AFFICHÉ : 13.70 €    │
│   MO unitaire brut = 13.65 ÷ 6 = 2.275 €                                    │
│                                                                             │
│ ─────────────────────────────────────────────────────────────────────────   │
│ ÉTAPE 3 : PRIME COST & FRAIS GÉNÉRAUX (FG)                                  │
│ ─────────────────────────────────────────────────────────────────────────   │
│   Prime Cost = 4.075 + 13.65 = 17.725 €                                     │
│   FG = 17.725 × 0.10 = 1.7725 €                      → AFFICHÉ : 1.80 €     │
│                                                                             │
│ ─────────────────────────────────────────────────────────────────────────   │
│ ÉTAPE 4 : COÛT COMPLET                                                      │
│ ─────────────────────────────────────────────────────────────────────────   │
│   Coût Complet total = 17.725 + 1.7725 = 19.4975 €                          │
│   Coût Complet unitaire brut = 19.4975 ÷ 6 = 3.249583 € → AFFICHÉ : 3.30 €  │
│                                                                             │
│ ─────────────────────────────────────────────────────────────────────────   │
│ ÉTAPE 5 : PRIX ACTUEL ET MARGE                                              │
│ ─────────────────────────────────────────────────────────────────────────   │
│   PV HT actuel = (4.075 ÷ 6) × 3.5 = 2.377083 €      → AFFICHÉ : 2.40 €     │
│   PV TTC actuel = 2.377083 × 1.10 = 2.614791 €       → AFFICHÉ : 2.70 €     │
│   Marge Nette réelle = ((2.377083 - 3.249583) ÷ 2.377083) × 100             │
│                      = -36.70%                       → AFFICHÉE : -36.7% ⚠️ │
│                                                                             │
│ ─────────────────────────────────────────────────────────────────────────   │
│ ÉTAPE 6 : RENTABILITÉ HORAIRE COMPLÉMENTAIRE                                │
│ ─────────────────────────────────────────────────────────────────────────   │
│   Profit total brut = (2.377083 - 3.249583) × 6 = -5.235 €                  │
│   Profit/heure brut = -5.235 ÷ 0.5 = -10.47 €/h      → AFFICHÉ : -10.50 €/h │
│                                                                             │
│ ─────────────────────────────────────────────────────────────────────────   │
│ ÉTAPE 7 : PRIX SUGGÉRÉ (Marge 25%)                                          │
│ ─────────────────────────────────────────────────────────────────────────   │
│   PV suggéré HT unitaire = 3.249583 ÷ 0.75 = 4.332777 € → AFFICHÉ : 4.40 €  │
│   PV suggéré TTC unitaire = 4.332777 × 1.10 = 4.766055 € → AFFICHÉ : 4.80 € │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ 6.2 - EXEMPLE COMPLET 2 : SAUCE TOMATE MAISON                               │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│ DONNÉES DE BASE :                                                           │
│   • Portions : 4  | Temps : 45 minutes (0.75 h) | Multiplicateur : 4.0      │
│                                                                             │
│ INGRÉDIENTS :                                                               │
│   • Tomate : 1000g à 0.0025 €/g = 2.50 €                                    │
│   • Oignon : 200g à 0.0013 €/g = 0.26 €                                     │
│   • Huile d'olive : 0.05L à 9.50 €/L = 0.475 €                              │
│   • Sel : 10g (prix unitaire null, ignoré)                                  │
│   • Ail : 20g à 0.005 €/g = 0.10 €                                          │
│                                                                             │
│ ─────────────────────────────────────────────────────────────────────────   │
│ ÉTAPE 1 : COÛT MATIÈRE (CM)                                                 │
│ ─────────────────────────────────────────────────────────────────────────   │
│   CM total brut = 2.50 + 0.26 + 0.475 + 0.10 = 3.335 € → AFFICHÉ : 3.40 €     │
│   CM unitaire brut = 3.335 ÷ 4 = 0.83375 €           → AFFICHÉ : 0.90 €     │
│                                                                             │
│ ─────────────────────────────────────────────────────────────────────────   │
│ ÉTAPE 2 : COÛT MAIN D'ŒUVRE (MO)                                            │
│ ─────────────────────────────────────────────────────────────────────────   │
│   MO total brut = (45 ÷ 60) × 27.30 = 20.475 €       → AFFICHÉ : 20.50 €    │
│   MO unitaire brut = 20.475 ÷ 4 = 5.11875 €                                 │
│                                                                             │
│ ─────────────────────────────────────────────────────────────────────────   │
│ ÉTAPE 3 : PRIME COST & FRAIS GÉNÉRAUX (FG)                                  │
│ ─────────────────────────────────────────────────────────────────────────   │
│   Prime Cost = 3.335 + 20.475 = 23.81 €                                     │
│   FG = 23.81 × 0.10 = 2.381 €                        → AFFICHÉ : 2.40 €     │
│                                                                             │
│ ─────────────────────────────────────────────────────────────────────────   │
│ ÉTAPE 4 : COÛT COMPLET                                                      │
│ ─────────────────────────────────────────────────────────────────────────   │
│   Coût Complet total = 23.81 + 2.381 = 26.191 €                             │
│   Coût Complet unitaire brut = 26.191 ÷ 4 = 6.54775 €    → AFFICHÉ : 6.60 €  │
│                                                                             │
│ ─────────────────────────────────────────────────────────────────────────   │
│ ÉTAPE 5 : PRIX ACTUEL ET MARGE                                              │
│ ─────────────────────────────────────────────────────────────────────────   │
│   PV HT actuel = (3.335 ÷ 4) × 4.0 = 3.335 €         → AFFICHÉ : 3.40 €     │
│   PV TTC actuel = 3.335 × 1.10 = 3.6685 €            → AFFICHÉ : 3.70 €     │
│   Marge Nette réelle = ((3.335 - 6.54775) ÷ 3.335) × 100                    │
│                      = -96.33%                       → AFFICHÉE : -96.3% ⚠️ │
│                                                                             │
│ ─────────────────────────────────────────────────────────────────────────   │
│ ÉTAPE 6 : RENTABILITÉ HORAIRE COMPLÉMENTAIRE                                │
│ ─────────────────────────────────────────────────────────────────────────   │
│   Profit total brut = (3.335 - 6.54775) × 4 = -12.851 €                     │
│   Profit/heure brut = -12.851 ÷ 0.75 = -17.13 €/h    → AFFICHÉ : -17.10 €/h │
│                                                                             │
│ ─────────────────────────────────────────────────────────────────────────   │
│ ÉTAPE 7 : PRIX SUGGÉRÉ (Marge 25%)                                          │
│ ─────────────────────────────────────────────────────────────────────────   │
│   PV suggéré HT unitaire = 6.54775 ÷ 0.75 = 8.730333 € → AFFICHÉ : 8.80 €   │
│   PV suggéré TTC unitaire = 8.730333 × 1.10 = 9.603366 € → AFFICHÉ : 9.70 € │
│                                                                             │

    GLOSSAIRE & RAPPELS
═══════════════════════════════════════════════════════════════════════════════

Coût matière (Food Cost)
  Coût total des ingrédients nécessaires à la production d'une recette.

Coût main d'œuvre (Labor Cost)
  Coût du temps de travail nécessaire, basé sur le taux horaire SAS chargé.

Charges patronales et salariales
  Cotisations globales appliquées pour le gérant de SAS (dirigeant assimilé-salarié),
  estimées à +82% du montant net souhaité (taux de 1.82).

Prime Cost
  Somme brute directe du Coût Matière et du Coût de Main d'œuvre (CM + MO).

Frais généraux (Overhead)
  Coûts indirects de production : électricité, eau, usure matériel, etc.,
  estimés à 10% du Prime Cost de la recette.

Coût complet (Full Cost)
  Coût total de revient comprenant le Prime Cost et les Frais généraux.

Prix HT (Hors Taxes)
  Prix de vente avant application de la TVA.

Prix TTC (Toutes Taxes Comprises)
  Prix de vente final incluant la TVA (HT × 1.10 pour TVA à 10%).

Marge nette
  Bénéfice de l'entreprise après déduction de toutes les charges.
  Calculée ainsi : ((PV HT - Coût Complet) ÷ PV HT) × 100.

Multiplicateur (Coefficient multiplicateur)
  Facteur appliqué au coût matière unitaire pour obtenir le prix de vente HT actuel.
  PV HT Actuel = (Coût Matière total / portions) × Multiplicateur.

Rentabilité horaire complémentaire
  Profit net d'entreprise dégagé par heure de cuisine. Permet de s'assurer
  que le salaire cible de 15 €/h net est atteint et dépassé (si > 0 €/h).

Allergène
  Substance alimentaire pouvant provoquer des réactions allergiques.
  14 allergènes majeurs sont réglementés par l'UE (Règlement 1169/2011).
